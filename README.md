# Project Portfolio
## Development
Start a web server by running the following command:
```
bin/dev
```
Database commands:
```
bin/rails db:drop
bin/rails db:create
bin/rails db:migrate
bin/rails db:seed
```

### Tailwind CSS
For styling this app uses [Tailwind CSS](https://github.com/rails/tailwindcss-rails).
```
bin/rails tailwindcss:build
```
